价值投资卖出策略

我目前已经简单学习了价值投资这个投资流派。并且已经执行了一年的对电力行业的投资，之前看好电力的理由有很多，大多数电力股票处于低估期，业绩好，分红高，容量电价等政策利好，算电协同政策等等

近一个月电力行业的股票涨得很快，我的票子基本上连续涨停了，但是我卖出的太早了，当时想的是跌下来后再接回来，这就导致我少赚了很多钱，大概30w。少赚这么多钱是很痛苦的事情，我好几天没有缓回来，有时候睡着了都会起来抽自己大嘴巴子

那么作为一个合格的价值投资者如何避免这种事情，同时对于这些股票来说，震荡行情也经常发生，如果一直拿在手里，收益并没有增加，如果我想做波段的话，也有卖飞的可能

因此问题变成了，作为价投者来说如何选择卖出的时机

# 心态

芒格对此有过精辟的论述：错过机会是投资过程的一部分，用不着太懊恼。 芒格自己也曾因一念之差错过贝尔里奇的股票，少赚了三四亿美元，他坦言"这是正常的，没什么大不了"。这是是每个价投者的必修课，在稳定赚钱之前，这些都是学费

更重要的是，芒格指出，大多数投资者"要么卖得太早，要么卖得太晚，而且永远是出于错误的原因"——因为股价下跌而卖、因为被电视上的大师煽动而卖、因为邻居买了加密货币赚钱而卖——这些理由没有一个与理性相关。我之前"想着跌下来后再接回来"的操作，本质上是在试图预测短期价格波动，这恰恰不是价值投资的思维方式

# 卖出时机的原则

巴菲特、本杰明和芒格等一流高手的卖出逻辑，和普通投资者完全不同。他们买入时采取的态度就像"买下的是一家私人企业"，着重于公司的长远经济前景、管理层以及支付的价格，从来没有考虑要在什么时候以何种价格再把这些股份卖出。只要预期内在价值能以满意的速率稳定增加，他们愿意无限期持有。正如芒格所说："如果你购买几个伟大的企业，那么你就可以安坐下来啦。那是很好的事情。"

那么，什么时候才应该卖出呢？巴菲特的卖出基于以下四个前提：

第一，公司基本面发生了根本性恶化。比如企业的护城河被侵蚀、管理层迷失了主要方向，或者公司的核心竞争优势不复存在。用巴菲特的话说，就是"如果一只股票不再符合他的某个投资标准"，他会毫不犹豫卖掉。巴菲特在疫情期间清仓航空股，就是因为疫情冲击超出了他对航空业长期前景的判断

第二，发现更优质的投资机会。当新发现的个股与现有持股一一比较，且明显更符合自己的投资哲学时，才进行替换

第三，买入本身就是个错误。随着深入研究发现当初的买入逻辑存在严重缺陷，这时应该果断纠正错误，而不是因为"套牢"而继续持有

第四，股价极度高估：这是巴菲特卖出决策的第三个理由，也是许多投资者关注的焦点。当一家公司的股价远高于其内在价值，市场报价严重透支了未来的增长潜力时，就是考虑卖出的时机。巴菲特在2007年清仓中国石油的案例充分说明了这一点：他在2003年以4.88亿美元买入中国石油的股票，当时其股价因油价上涨和市场追捧而高企，市值达到2750亿美元，市盈率远超同行。巴菲特判断，此时股价已充分甚至过度反映了公司的价值，继续持有的安全边际已荡然无存。于是，他选择在股价高位套现，获得了约7倍的收益。同样，在2022-2023年，当苹果公司的市盈率突破30倍，远高于巴菲特偏爱的15倍以下标准时，他选择高位减持苹果股票，锁定巨额收益。需要强调的是，巴菲特对于拥有宽阔护城河的伟大公司，在极度高估时只会减持，不会清仓，永远保留底仓。这意味着，即使在估值泡沫化时卖出部分获利，他仍会保留核心仓位，以长期分享企业成长的成果。

关键错误操作是单纯因为"股价涨高了"就卖出。以巴菲特为代表的价值投资者，更愿意陪伴优秀企业长跑，更关注企业基本面是否发生变化，一般不会因高估而卖出。李录（被芒格称为"中国的巴菲特"）曾说：当他对某个领域产生洞见时，即便有人说价格已经很高了，而且确实自己也不愿意再买，但从长期来看，他对这家公司的深刻理解告诉他赢面还是很大，"到了这时候，你根本不会想卖"

而芒格则偏向于尽量减少卖出决策，“如果你是因为低估而买入，那么当股价接近你计算的内在价值时，你就得考虑卖出。这很难。但如果你买的是几家极其优秀的公司，那你就可以屁股坐着不动，这才是好事。”

拆开就是两层：

1，从“低估”走向“合理”：价投赚的是“价格向价值回归”的钱，回归得差不多了，就该考虑卖，否则就变成投机 2，从“烟蒂股”走向“伟大公司”：如果你买的是少数真正伟大的公司，而且买得不贵，那尽量减少卖出决策，长期持有是更优策略。因为卖得越勤，犯“换到更差标的”的错误概率越大

芒格自己也说，他更愿意“在少数几次机会下重注，然后长期坐着不动”，而不是频繁交易。这和目前我的投资风格符合，满仓满融单吊红利

而格雷厄姆强调要有明确的“卖出政策”，不是看心情，格雷厄姆在《聪明的投资者》里很明确地说：

“投资者应该对所有普通股投资制定明确的卖出政策，与他的买入策略相匹配。通常，他应该为每次买入设定一个合理的利润目标——比如 50%—100%——以及实现该目标的最长持有期——比如 2—3 年。在持有期末未能实现利润目标的买入，应该按市价卖出。”

核心思想是买入时就要想好什么价格/什么条件下卖，而不是涨了再拍脑袋。他达到目标收益、超过时间还没兑现，就先出来，而不是死扛。反人性的机械式操作决定了他的纪律性，这也是我目前要学习的地方。同时一些短线打法能赚钱，很可能是因为操作者为短线打法设定了一套完整的买入卖出方案，并且严格执行，这才能赚钱，因为市场是无法预测的，无论是 macd、量级、金叉死叉等等这些指标，也就是反应了市场过去一段时间的走势，未来是无法预测的

# 估值锚定方法

内在价值是唯一的锚

巴菲特的估值哲学浓缩为四个关键词："能力圈"、"逆向投资"、"安全边际"、"长期持有"。其中估值层面的核心工具是自由现金流折现模型（DCF）——"公司现在的内在价值 = 未来每年能自由支配的现金流，按一定折现率折算成今天的总和"。它通过预测企业未来产生的自由现金流，并以适当的折现率将其折现回现值，得出企业的内在价值。巴菲特和芒格都认可 DCF 的逻辑，认为一家公司的内在价值等于其未来生命周期内所能产生的自由现金流的折现值。然而，DCF 模型的复杂性和对未来的高度不确定性，使得普通投资者难以准确应用

在实际应用中，巴菲特自己也坦言不会拿着计算器精确计算内在价值，他更强调的是"一眼就能看出来很值"的模糊正确。芒格的投资评估过程也是如此：最重要的观念是把股票当成企业的所有权，根据竞争优势来判断企业的持有价值，如果企业未来的贴现现金流比现在购买的股票价格要高，那么这个企业就具有投资价值

DCF 的运用更像是一种严谨的思维框架，用来逼迫你透彻理解一家公司的生意模式、盈利驱动和风险因子，而不是一套可以机械代入的数字游戏，不是计算、不是计算、不是计算

有些人将企业的估值状态分为三个区间，即三区间估值框架：

1，低估区间：股价低于内在价值，此时买入有安全边际保护。巴菲特追求的是"用40美分购买价值1美元的股票"

2，合理区间：股价大致匹配内在价值，虽然没有安全边际，但持有依然可以享受企业成长带来的回报。此时核心关注的是分红收益和企业内在价值的增长速率

3，高估区间：股价超过内在价值，安全边际消失，追高风险显著加大。如果价格"远高于"价值，且找不到合理的未来增长来支撑当前价格，才考虑卖出

根据巴芒体系的"凯利公式导向"和"集中投资原则"，仓位配置取决于确定性和赔率。芒格认为当一只股票的确定性足够高时，仓位可以适度集中，分散投资反而会稀释优势。因此我的投资策略为满仓单吊，在保证仓位设计如下：

1，基本仓（核心底仓），总资产的 66%，除非基本面逻辑彻底破坏或出现不可逆的恶化，否则永不卖出， 低估区建仓完毕，合理区和高估区均持股不动 2，波动仓（机动仓位），总资产的 33%，仅在低估区间分批买入，高估区间分批卖出，合理区呆坐不动

那具体怎么估呢？将价格划分为三个区间事实上最难的部分是找到正确的区间值，区间值则看你对企业的理解程度，普通人很难算准，就算接下来我将介绍 DCF 的大致流程，这是为了帮助我们算出一个估值，这个估值很可能不准，但是没关系，因为我们的目标是根据区间，而不是具体的数值

# DCF

大致流程：做 3–5 年的粗略现金流预测 + 简化折现率 + 永续增长 0–2%，得出一个价值区间，再结合 PE/PB 等指标一起判断

# 总结

卖出时机的原则： 1，公司基本面发生了根本性恶化。 2，发现更优质的投资机会。 3，买入本身就是个错误。 4，股价超过内在价值，且找不到合理的未来增长来支撑当前价格。
