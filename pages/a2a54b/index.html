股票技术面——基础

关注资金入场意愿是否强烈，以找到合适的时机入场。那这里的入场意愿是否强烈该如何判断呢

技术面分析可以作为买卖时机判定的工具，但是投资还是需要基本面的分析作为支撑。也就是基本面选股，技术面判断买点卖点

# 均线

均线，全称为移动平均线（Moving Average），英文简称为 MA，该指标是以平均成本概念为理论基础，采用统计学中移动平均的原理，将一定时间周期内的股价或指数的平均值标在价格图表中连成曲线，用来显示股价或指数的历史波动情况，并以此来预测后市的趋势走向，为投资者提供操作依据

在一般的炒股软件中，会展示多条均线，5日、10日、30日、60日等，这些均线会用 x 天内收盘价的平均值，连成一条线画在 K 线图上

短期均线中最常用的是以5日均线和10日均线，分别代表一周或两周的平均价。短期均线揭示了市场的短期震荡，投资者可以以此做出短线买卖的依据

中期均线中最常用的是以20日、30日和60日为计算周期，20日均线代表一个月（4周）的平均股价，在中短线操作时常会被用到；30日均线和60日均线（季线）的波动更具稳定性，能指出了市场的中期波动方向，是投资者中线操作的重要依据

下面讲一下均线的常见用法

# 观察趋势

均线能够代表趋势，但是不能预测趋势，趋势方向是均线的第一要素。均线通常有两种方向，一种是向上，另一种是向下。均线的方向与历史数据有关，如果均线的当前值比前一阶段的均值大，那么均线的方向是向上的；如果均线的当前值比前一阶段的均值小，那么均线的方向是向下的。当中长期均线向上运行时，表示中长期趋势以上涨行情为主，此间出现的股价下跌通常是短期回调；当中长期均线向下运行时，表示中长期趋势以下跌行情为主，此间出现的股价上涨通常是短期反弹

K线位于均线的上方，则表示股价高于均线对应周期内的大多数投资者的持仓成本，也就意味着股价上涨相对比较容易；反之，则下跌比较容易，这里涉及到投机之王利弗莫尔操盘核心，沿最小阻力路径操作，下文再讲

# 判断空方多方力量

均线的拐点是指在该点前后，均线的运行方向发生了相反的变化。当一根中长期均线向上运行了很长时间之后，股价无力再创新高而出现下跌，均线也随之移动，进而形成了一个波峰，这种拐点预示着原来的上升趋势可能转变为下降趋势

当一根中长期均线向下运行了很长时间之后，股价无法再创新低而出现上涨，均线也随之移动，进而形成了一个波谷，这种拐点预示着原来的下降趋势可能转变为上升趋势。中长期均线的拐点有较高的参考价值，可使投资者有效地掌握股票买卖的最佳时机。而短期均线的拐点较多，可信度也相对较低

当短日线上穿长日线的时候，被称作金叉，一般代表短期内多方力量强劲，是适合买入的时机。当短日线下传长日线的时候，被称作死叉，一般代表短期内空方力量强劲，是适合卖出的时机

这里说的就是金叉死叉交易策略，而上面的短日线和长日线又可以是5日均线和10日均线，可以是60分线和10日均线，这里的交易策略可能在一个行情一个股票中可能亏钱，但是在另外一个行情和另外一个股票中可以赚钱，有些庄家也会在均线附近，故意大量卖出股票造成恐慌盘，以吸收低价筹码，因此均线交易策略也可能完全是错误的，摸索出那些股票适合使用那些交易策略，是新手进阶的第一步

# 压力位和支撑位

均线附近可能会有压力位和支撑位。即这些位置会有一定的助长助跌性，也可能没有，主要看市场和该标的对这条线认不认可，如果不认可就没有

压力位通俗点来说是指大伙都是在价位被套的，如果说股价上涨到某价位的附近了，股票停止上涨， 甚至会出现回落的情况。这个起着阻止或暂时阻止股价继续上升的价位就是压力位

支撑位是指大伙对于这个股票的估值是什么价位，觉得什么价位买入比较能赚钱。在股价跌到某个价位附近的情况下，股价将不会继续下跌，甚至还有回升的可能性，支撑位是一种价格——阻止股价继续下跌或暂缓股价继续下跌

压力位就是阻力，任何时候都有阻力，上涨中也有阻力，下跌中也有。但是上升过程的压力位是会导致震荡上升，下跌过程的压力位是震荡下跌

对于市场中的人们来说，历史高点或者历史低点都有可能生成压力位或者支撑位，毕竟大家都会觉得历史的数据比较有参考意义。但是这种说法其实没啥根据，历史虽然每天都在重演，但是也不太可能发生和明天一模一样的事情

更加具有依据性的数据是30日均线和60日均线，可以让你知道大多数人是在什么价位购买的这只股票，在这个价格附近多半是压力位或者支撑位了，但是也有可能出现方差特别大的情况导致数据不准确

更更加具有依据性的数据就是其他用户的具体购买数据，不过这个需要靠平台来给你提供了，至于准不准，谁知道呢

# MACD

MACD 称为异同移动平均线，是从双指数移动平均线发展而来的，由快的指数移动平均线（EMA12）减去慢的指数移动平均线（EMA26）得到快线 DIF，再用2×（快线 DIF-DIF 的9日加权移动均线 DEA）得到 MACD 柱。MACD 的意义和双移动平均线基本相同，即由快、慢均线的离散、聚合表征当前的多空状态和股价可能的发展变化趋势，但阅读起来更方便。MACD 的变化代表着市场趋势的变化，不同 K 线级别的 MACD 代表当前级别周期中的买卖趋势

MACD 核心还是均线，不过不是上文中说到的均线，EMA 是指加权移动平均线，简单来说，还是 x 天内收盘价的平均值，但是离本日越近的收盘价，会分配更多的权重。因此比起均线，MACD 会更加侧重短线操作

DIF 是字面含义，快线减慢线，DIF 为正说明是上涨行情，DIF 为负则是下跌行情。两根均线的差离值越大，DIF 也就越大

慢线 DEA 为9天内，快线 DIF 的平均值

故 MACD 指标是由两线一柱组合起来形成，快速线（白色线）为 DIF，慢速线（黄色线）为 DEA，柱状图为 MACD。柱状图计算方式为 （DIF-DEA）×2，如果在正说明短期行情上行，如果为负说明短期行情下行

# 洗盘

洗盘就是市场在连续上涨之后，需要一个：

1，主力消化获利的过程 2，回吐盘的压力，即之前在这个价格买入股票的玩家，觉得股票终于回到这个价位了，因此赶紧卖出股票解套，这就叫压力

主力机构利用长阴刻意制造空头气氛（就是让股票一直跌），使市场投资者产生恐慌心理，间接给市场已经到了头部的假象，让获利有兑现要求的投资者交出自己的筹码

这种洗盘行为往往是建立在市场以及股票在刚刚启动初期，在上涨一段之后的行为。可视为主力洗盘式吃货行为。洗盘就是为吃更多货，收集更多的筹码，并且把散户清理出去

有时候洗盘可能转换为出货，有时候出货可能转换为洗盘，主要看洗盘期间拿到的筹码多不多，这都是由主力控制的，而我们散户只能去博弈主力的动向

方式：

  • 直接打压式：直接打压法较多出现在庄家吸货区域，目的是为了吓退同一成本的浮动筹码。在盘中表现为开盘出奇地高，只有少许几笔主动性买单便掉头直下，一直打到前一日收盘价之下，持仓散户纷纷逢高出局

  • 横向整理式：采用横向整理洗盘的主力实力是较弱的，往往保持一定幅度的震荡，在震荡中不断以低吸高抛赚取差价以摊低成本和维持日常的开支。实力较强的主力，往往将股价震幅控制在很窄的范围内，并且拉低成交量，使其走势极其沉闷，让散户将筹码转出去抄其他股，以达到吸筹的目的，这种方法好像被以前的朋友叫做青龙取水

青龙取水的形态特征：在5日、10日和60日均线图上，5日均线和10日均线不断地上下起伏，形如龙体。日K线图依附在龙体上如片片龙鳞，如此形成的青龙逶迤在60日均线之上。然后龙头回到60日均线附近，犹如青龙入河取水。同时成交量会极度萎缩。有时候横盘的时间会非常长，如果看好的话需要很大耐心

但是这么拽的名字很可能在实际分析没有任何用，有可能长时间横盘后继续横盘或者大跌，也有可能长时间横盘后发现根本没有主力只有散户在这个票子里玩，所以了解一下就好了

# 选股

一进二选股需要遵守以下原则，优选如下：

1，近一个月有过涨停，近日有涨停并且回踩不破涨停起始位（确保资金已入场，并且资金看多） 2，近期无明显异动放量（指下跌放量，这种情况说明出货），首版倍量最佳（资金入场，突破验证），缩量则需要注意风险（第二天可能会出现分歧） 3，次新叠加近期热点最佳（炒新不抄旧，一般次新没有庄家，新资金入场怕庄家，因为新资金进来把价格抬上去了，这时候庄家可以先出货，或者压价。近期热点则保证了可以吸引新资金） 4，低位首个涨停板最佳（无高位套牢风险） 5，流通市值小于50亿，股价低于20块（柚子比较好拉升） 6，无开通融券功能（无法做空）

不选：

1，一字板不选（过强的，起一字版不想给人进场机会，起完之后给人进场机会的话就是不伦不类，可能是要换手了） 2，一年出现大幅连板回落的不选（前一波的套牢的资金可能没出完） 3，一年内一直跌，并且处于历史低位，这时候涨停了，不选（有稳定下跌趋势，一个板就流转下跌趋势的话，概率不大） 4，近期被均线压制，涨停没能突破压制的不选（套牢盘太多） 5，流通盘大于200亿不选（资金量太大，不好拉升）

# 买点

第一天晚上选好股票之后，在集合竞价时就需要注意观察买点了：

1，板块观察，观察个股所属板块的日内强度 2，竞价观察，重视25分钟，此时如果有大买单重点观察。如果卖单比较多可能今天就封不住板，因此这些票子淘汰掉（强单竞价直接买入在行情好的时候可能好一点，弱单也是；但是太弱了就不要买了） 3，做好涨停挂单准备 4，预防骗线，看挂单、成交明细、分时量价、macd

# 离场方法

1，三版炸板直接离场（也需要看情绪和政策） 2，小高开冲高7-8点回落离场（因为这个票很强，那就不应该给机会让人参与，要是回落了就说明分歧了，熊市赶紧跑，牛市看资金博弈） 3，小低开，往上走，盘中大卖单出现时离场，不要逗留（分歧） 4，二板失败，隔天竞价不强直接走，千万不要留（既然要玩这种票，止损就一定要快） 5，二板失败，隔天竞价较强，往上走时参考小低开离场

柚子散户是对手盘，假如，使用1进2方法的人太多，柚子就会故意炸板制造假象
