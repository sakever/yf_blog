梅花易数学习笔记

# 前置知识

# 什么是八卦

《太平御览》：“伏羲坐于方坛之上，听八风之气，乃画八卦。”

《周易·系辞》：“易有太极，是生两仪，两仪生四象，四象生八卦。”

我们可以看到两仪包括“—”和“--”，“—”称之为阳爻（yáo），“--”称之为阴爻，他们排列组合组成八卦。

每一挂代表一种自然现象：乾为天，坤为地，震为雷，巽（xùn）为风，坎为水，艮（gèn）为山、离为火，兑为泽。并且根据这些基础的现象，可演化出万事万物，某一挂所演化出的现象可以被该挂所代表，称之为万物类象。

有一个口诀可以帮助记住这些卦象：

乾三连（☰），坤六断（☷）；震仰盂（☳），艮覆碗（☶）；

离中虚（☲），坎中满（☵）；兑上缺（☱），巽下断（☴）。

# 八卦的顺序

八卦的排列分先天八卦和后天八卦。

先天八卦又称伏羲八卦。它主要用来描述自然，把八卦代表的天地风雷，山泽水火八类物象分为四组，以说明它的阴阳对峙关系。

下图这些数字代表了先天八卦的顺序，先天八卦凡是相对的两个卦的数字加起来，都等于九。

先天八卦数是：乾一、兑二、离三、震四、巽五、坎六、艮七、坤八。相传来自于河图

后天八卦又称文王八卦。后天图是从四时的推移，万物的生长收藏得出的规律。

可以把这个后天八卦，放到左手的手指上，无名指的根节上放乾卦，中指的根节上放坎卦，食指的根节放艮卦，食指的中节放震卦，食指的尖节放巽卦，中指的尖节放离卦，无名指的尖节放坤卦，无名指的中节放兑卦。

同理，先天八卦也可以放上去，不过是顺序不同。

除此之外还有一个口诀应该记住：

一数坎兮二数坤，三震四巽数中分，五寄中宫六乾是，七兑八艮九离门。

这个口诀来自洛书的九宫格分布，这些数字代表了后天八卦的顺序，凡是相对的两个卦的数字加起来，都等于十。

# 梅花易数

# 起卦原则

梅花易数名称的来源很有意思，说是在北宋，有一位伟大的哲学家叫邵雍。有一天，这位先生进入梅花园赏花，突然看见两只麻雀在枝头上争吵。先生认为不动不占，立即据此推算，得出次日傍晚邻居家的女儿上树折花、摔伤其腿的结论。事后果然应验，其他人为此卦例特殊，竟能断出与卦题不相干之事情来，将此种断法命名为“梅花易数”。

起卦的原则是无事不占，不动不占，不为同一事反复占问。

《告别娑婆2断轮回》：你所看到的一切都是你内心投射的结果，并非真实。

《奇迹课程》：投射形成知见。你眼中的世界，全是你自己赋予的，如此而已，既不多，也不少。因此，它对你变得十分重要。它是你心境的见证，是描述你内心状态的外在表相。

当外部环境的变化，也许是心境的变化，所以不动不占，这也许也是古人们的哲学吧。

# 如何起卦

可以使用年月日时起卦：

以农历之年月日总和除以八（因为挂象总共有八个），以余数为卦数求上卦;以年月日时总和除以八，以余数为卦数求下卦，再以年月日时总和除以六（六十四挂每挂由六个爻组成），以余数为动爻。

这里有几点细节：

年所代表的数是阴历地支的数字，顺序是：

子1、丑2、寅3、卯4、辰5、巳6、午7、未8、申9、酉10、戌11、亥12

比如今年是鼠年，所代表的数就是1，今年是龙年，所代表的数就是5

得出的数字用先天八卦顺序算出挂，比如余数为3，得出的挂应该是离挂，而不是震。

也可以不使用时间起卦，直接用数字起：可以随意说出两个数，第一个数取为上卦，第二个数取为下卦，两数之和除以6，余数为动爻。

可以按字的划数或者字数起卦。字少时，,按笔画数，字多时，可用字数起卦。

以此类推，大多可以得出上下挂和动爻这三个数字的行为，都可以用来起卦。

# 如何断挂

上面说了一个新名词：动爻。得出的挂象，从下向上数，动爻是几就改变第几位的爻。

动爻是区别和确定体卦、用卦的主要标志，一卦有上下之分，有动爻的卦为用卦，无动爻的卦为体卦，动爻在下卦者，下卦为用。动爻在上卦者，上卦为用，体卦为自己，用卦为他人或事。将自身设为参照物，自己是不变的，其他万物都会发生改变。

根据八卦对应的五行得出体用关系，一般来说： 用克体，诸事难为，大凶，表示事情不会成功。 用生体，进益之喜，大吉，表示事情不费力便可成功。 体生用，耗劳之忧，小凶，表示费力而难以完成。 体克用，得劳兼备，小吉，表示事情的进展比较费力。 体用比合(体和用五行相同)，诸事顺利，吉，表示进展顺利。 当五行没有生克关系时，根据挂象来分析凶吉。
